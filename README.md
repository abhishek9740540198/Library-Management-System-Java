# Library Management System (Java)

A simple console-based Library Management System developed in Java.

## Features
- Add a Book
- Display All Books
- Search a Book
- Delete a Book

## Technologies Used
- Java
- ArrayList
- Scanner

## Files
- `LibrarySystem.java` – Main version with formatted menu.
- `WebLibrarySystem.java` – Alternative implementation.

## How to Run

Compile:
```bash
javac LibrarySystem.java
```

Run:
```bash
java LibrarySystem
```
